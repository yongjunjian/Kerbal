◆ operator^()

const Matrix kerbal::math::operator^ ( const Matrix &  A,
const int  n 
)
throw (std::invalid_argument
)
计算规则
  • 若 n 为负数, 则计算基数矩阵的逆矩阵的 -n 次幂
  • 若 n 为 0, 则返回同样大小的单位矩阵
  • 若 n 为 1, 则返回原矩阵的拷贝
参数
A基数矩阵
n次数
返回
A 的 n 次幂
异常
std::invalid_argument当 n 不等于 1 时, 要求基数矩阵必须为方阵, 否则抛出此异常
备注
表达式 A ^ n 完全等效于函数表达式 pow(A, n)
参见
const Matrix pow(const Matrix & A, const int n )
