Detailed Description

template<class Type>
class kerbal::data_struct::array_2d::Array_2d< Type >

动态二维数组类

Author
倪文卿
Remarks
本类提供了对动态二维数组的支持, 相比于 std::vector<std::vector<Type> > , 此类可确保二维数组的每一行拥有相同的元素个数, 且更节省存储空间
