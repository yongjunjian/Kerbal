详细描述

template<typename Tp, size_t N>
class kerbal::data_struct::static_stack< Tp, N >

Stack with flexible length that stored on automatic storage duration

The class is an encapsulation class of stack that could be stored on automatic storage duration. It is more effective than std::stack because std::stack store elements on heap storage duration (that's required memory allocate and deallocate).

模板参数
TpType of the elements.
NThe maximum number of elements that the stack can hold.
