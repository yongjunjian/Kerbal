◆ operator^()

const Matrix kerbal::math::matrix::Matrix::operator^ ( int  n) const

计算矩阵的 n 次幂

计算规则
  • 若 n 为负数, 则计算基数矩阵的逆矩阵的 -n 次幂
  • 若 n 为 0, 则返回同样大小的单位矩阵
  • 若 n 为 1, 则返回原矩阵的拷贝
Parameters
n次数
Returns
A 的 n 次幂
Exceptions
std::invalid_argument当 n 不等于 1 时, 要求基数矩阵必须为方阵, 否则抛出此异常
Remarks
表达式 A ^ n 完全等效于函数表达式 pow(A, n)
See also
friend inline const Matrix pow(const Matrix &A, int n)

计算一个矩阵的 n 次幂

计算规则
  • 若 n 为负数, 则计算基数矩阵的逆矩阵的 -n 次幂
  • 若 n 为 0, 则返回同样大小的单位矩阵
  • 若 n 为 1, 则返回原矩阵的拷贝

A基数矩阵
n次数
Returns
A 的 n 次幂
Exceptions
std::invalid_argument当 n 不等于 1 时, 要求基数矩阵必须为方阵, 否则抛出此异常
Remarks
函数表达式 pow(A, n) 完全等效于表达式 A ^ n
